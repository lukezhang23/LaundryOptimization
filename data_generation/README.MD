# Data Generation

## Table of Contents
- #### seed
  - #### length_weights.csv: Weights for generating duration of availability window
  - #### names.csv: List of names to generate data for
  - #### start_time_weights.csv: Weights for generating start time
- #### generate_chart.py: Creates schedule_chart.pdf off of data.csv
- #### generate_data.py: Creates random data in data.csv for the names listed in seed/names.csv
- #### initalize_project.py: Runs all the initialization scripts in the proper order.
- #### pdf_to_png.py: Creates schedule_chart.png from schedule_chart.pdf for the root README
